🤖 AI-Powered Customer Support Intelligence Platform

🚀 Final Project Report

📌 Project Overview

This project builds an intelligent AI system to automate customer support ticket analysis using NLP and Machine Learning.

  • 🔍 Ticket Classification
  • ⚡ Priority Prediction
  • ⏱️ Resolution Time Prediction
  • 📊 Customer Segmentation

📅 Day 1: Data Cleaning

  • 🧹 Removed irrelevant columns
  • ⚠️ Handled missing values
  • 📅 Date feature extraction
  • 📏 Created text length features

📊 Day 2: EDA

  • ⚖️ Balanced dataset
  • 📉 Resolution time mostly 0–10 hrs
  • 📡 Multi-channel support usage
  • ⭐ Mixed customer satisfaction

🧠 Day 3: NLP Preprocessing

  • 🔡 Lowercasing + cleaning
  • 🚫 Stopwords removal
  • 📚 Lemmatization
  • 🔗 Created final_text
  • 📊 TF-IDF (5000 features)

🤖 Day 4-5: ML Models

  • 📉 Logistic Regression (~20%)
  • 📈 Naive Bayes (~24.5%) ✅ Best
  • 🌳 Random Forest (~23%)
⚠️ Insight: Dataset is repetitive → limits performance

🔥 Day 7: DistilBERT

  • 🚀 Transformer-based model
  • 📊 F1 Score ≈ 0.18–0.20
  • 📉 Confusion Matrix created
💡 Insight: Handles context better than ML models

📊 Day 8: Regression & Clustering

⏱️ Regression

  • 📉 RMSE ≈ 26
  • ❌ Negative R²
⚠️ Text alone is not enough for regression

🧩 Clustering

  • 🔵 Cluster 1: Simple issues
  • 🟡 Cluster 2: Moderate issues
  • 🔴 Cluster 3: Complex issues

💡 Business Insights

  • 🛠️ Technical issues take longer
  • 💳 Billing resolved faster
  • 📊 Customer issues vary in complexity
  • 🤔 Text data is ambiguous

🏁 Final Conclusion

The system successfully demonstrates an end-to-end AI solution for customer support automation. DistilBERT proved to be the strongest model.
